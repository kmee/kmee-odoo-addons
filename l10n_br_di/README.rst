Configuration
=============

Acesse o cadastro da empresa e configure a operação fiscal padrão para
geração das DIs.

Usage
=====

Para usar esse módulo você precisa:

::

   1. Importar ou criar uma DI manualmente;
   2. Gerar uma fatura em rascunho;
   3. Revisar a transmiti-la.
