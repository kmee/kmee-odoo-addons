When an inbound transfer is created, "Marked as To Do" and the contact
is a multi-company company, in that company, a outbound transfer is
created.

Configuration
=============

Follow the steps in stock_intercompany_bidirectional readme.

Usage
=====

Create a receipt transfer and check if at the other company a delivery
transfer was created.
