Usage
=====

To use this module you need to:

::

   1. Go to applications form view.
   2. Click 'Refuse'.
   3. Select a reason.
   4. Select 'Send Mail'.
   5. You are able to edit the mail clicking the button on the bottom of the view 'Edit Mail'.
